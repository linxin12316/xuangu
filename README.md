# 选股 (xuangu)

A 股每日选股工具：每个交易日盘前 08:30 + 盘后 16:05 自动推送到你的微信。

> ⚠️ **严肃声明**
> - 本工具只是**量化筛选辅助**，不是"必涨预言"。胜率上限约 55%。
> - 推送的 3-5 只候选**仅供研究**，请自行判断、严格止损、控制仓位。
> - 投资有风险，亏损自负。任何宣称"必涨""稳赚"的工具都是骗子。

---

## 它做什么

每个交易日：

| 时间 | 任务 | 输出 |
|---|---|---|
| 08:30 | 盘前选股 | 强势板块 Top 5 + 候选股 Top 5（含五维评分、止损建议） |
| 16:05 | 盘后复盘 | 大盘总结 + 候选股当日表现 + 命中率 |

选股逻辑全部透明，详见 [选股算法](#选股算法)。

## 一次性配置（5 分钟）

### 步骤 1：注册 Server 酱（微信推送通道）

1. 打开 https://sct.ftqq.com
2. 微信扫码登录
3. 进入「SendKey」页面，复制你的 SendKey（形如 `SCT123456...`）

### 步骤 2：填入仓库 Secret

1. 打开 https://github.com/linxin12316/xuangu/settings/secrets/actions
2. 点击 **New repository secret**
3. Name 填 `SCKEY`，Value 粘贴上一步的 SendKey
4. 点击 **Add secret**

### 步骤 3：手动触发一次验证

1. 打开 https://github.com/linxin12316/xuangu/actions
2. 左侧点 **Daily Stock Pick**
3. 右上角点 **Run workflow** → 绿色按钮
4. 等待 2-5 分钟，刷新页面看是否绿勾
5. 微信收到一条「📈 选股报告」= 配置完成 ✅

之后**每个交易日**早 8:30 / 晚 16:05 会自动推送，你什么都不用做。

## 选股算法

### 阶段 1：剔除地雷股
- ST / *ST / 退市股
- 当日停牌
- 总市值 < 30 亿
- 北交所（流动性差）

### 阶段 2：板块择强
- 计算所有申万一级行业近 5 日涨幅
- 取 Top 5 板块的全部成分股作为候选池

### 阶段 3：五维打分（满分 100）

| 维度 | 权重 | 含义 |
|---|---|---|
| 趋势 | 30 | MA5/10/20 多头排列 |
| 量能 | 25 | 近 5 日相对前 20 日的放量倍数 |
| 动量 | 20 | RSI(14) 在健康区间（50-70 满分，>80 归零防追高） |
| 资金 | 15 | 北向资金近 5 日持股变动 |
| 安全 | 10 | 距 60 日均线偏离度（防止追高） |

**取分数最高的 5 只 → 推送。**

### 阶段 4：止损建议

每只候选给出一个止损价 = `max(MA20, 现价 × 0.93)`，跌破立刻离场。

## 使用纪律（重要）

1. 同一板块**最多持有 2 只**，避免过度集中
2. 单只仓位**不超过总资金 20%**
3. **跌破止损价立即离场**，不抗单
4. 综合分 < 70 的候选**优先级降低**
5. 标 `🔁 已连续上榜 N 日` 的股票要警惕已经涨过头
6. 大盘单日跌 > 1.5% 时**主动回避**当日候选

## FAQ

**Q: 为什么不直接选 1 只必涨的？**  
A: 没人能做到。即使顶级量化基金次日胜率也仅 52-55%，集中买 1 只 ≈ 抛硬币。

**Q: 推送的候选我必须买吗？**  
A: 不必。这是**研究清单**，不是买入清单。你应该结合自己对个股的认知再决定。

**Q: 工具哪天会失效？**  
A: 当主流量化资金涌入这套规则时，超额收益会被吃掉。这套策略**简单、透明**，不指望永远赚钱。

**Q: 我能改算法吗？**  
A: 可以。修改 `src/scoring.py` 中的权重和规则，commit 推送即可，下次 Action 自动用新版本。

## 撤销 Token（重要安全操作）

本仓库由 Claude Code 创建时使用的 Personal Access Token 已在历史会话中暴露。  
**强烈建议**完成首次跑通后立即去 https://github.com/settings/tokens **revoke 旧 Token**，  
重新生成一个新的，且不要再保存到任何 AI 助手的记忆中。
